# 建筑板部工具箱

把「建筑板 / 冷库板」项目概览表的整理工作自动化的桌面小工具（Windows / macOS）。

把**装箱明细、成品物流、切锯表、下单详情**这些表丢进去，一键生成或更新带「汇总」「每日发货」的项目概览表，省掉手工合并、核对、统计的反复劳动。所有数据全程在本机处理，不上传任何服务器。

## 快速上手

1. **左侧选一张卡片**：建筑板 · 概览表 / 冷库板 · 概览表。
2. **准备文件**：把对应的表**拖进**文件槽，或点击文件槽选择。每个槽标了「必填 / 可选」。也可以用下方的 **从系统拉数** 自动拉取（见下）。
3. **不上传概览表**：在顶部填**项目名**、选**保存目录**生成新文件。**上传概览表**：直接在你选的那份概览表上更新，不用选目录。
4. 点 **执行**（或按 `Ctrl/⌘ + Enter`）。
5. 完成后点 **打开文件 / 打开目录** 查看结果。右侧「事件」区有每一步的回执，出问题会写明原因。

支持的表格格式：`.xlsx` / `.xlsm`；下单详情额外支持 `.csv`（UTF-8 或 GBK 编码）。

## 两张业务入口怎么用

| 入口 | 自动派生 | 需要的表 |
|---|---|---|
| **建筑板 · 概览表** | 不上传概览表：生成；上传概览表：更新 | 概览表（更新时）、装箱明细（生成必填，可多份）、成品物流、切锯表、下单详情 |
| **冷库板 · 概览表** | 不上传概览表：生成；上传概览表：更新 | 概览表（更新时）、下单详情（生成必填）、成品物流、切锯表 |

> **更新模式**：传哪张表就刷新哪一块；都不传则只重算「汇总」和「每日发货」。

## 从系统拉数

不用手动从生产管理系统导出再拖进来。填好 **下单详情**（或已有 **概览表**）后，点 **从系统拉数**，工具会按其中的库号、项目名自动拉取装箱明细（仅建筑板）、切锯表、成品物流，并填进对应文件槽。

- 需要联网到公司生产管理系统（查询免登录）。
- **「高级」可调**：车间（默认精工连续线）、切锯范围（近 1 / 3 / 6 / 12 月）、物流时间窗（单次最多查近 180 天，跨度更长会自动分两段查询再合并去重）。
- **一次只处理一个项目**：下单详情 / 概览表里若混了多个项目，会直接提示并停止——请删掉多余项目的行再拉，避免拉到别的项目的数据。

## 选项说明

- **汇总**：在概览表最前加「汇总」sheet，按 板宽＋颜色＋板厚（冷库板按 外板颜色＋内板颜色）统计各状态面积。
- **每日发货**：加「每日发货数据」sheet，按出库日期统计发货面积。
- **跳过 Excel**：不调用本机 Excel 做原生重算（之后用 Excel 打开结果会自动重算）。检测到本机装有 Microsoft Excel 时默认走原生重算，面积 / 汇总公式即时算好。
- **先备份**（仅更新模式）：改动前先备份原概览表。

## 自动更新

启动时自动检查新版本（COS / R2 / GitHub 三通道兜底，哪个通用哪个）。安装版走安装包更新；也可在「关于」里手动检查。

## 隐私

所有表格都在你的电脑上处理，**不上传服务器**。工具只在本地记录最近任务的文件名等元信息，从不复制或缓存你的业务文件内容。
